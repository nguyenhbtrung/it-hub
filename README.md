## Overview

IT Hub is a modern E-Learning platform designed for IT students, instructors, and administrators.

The platform provides:

* Course discovery and enrollment
* Structured learning paths with lessons and exercises
* Progress tracking and learning analytics
* Instructor course creation and management
* Administration and content moderation tools
* AI-powered learning assistant
* File upload and content delivery infrastructure

The system consists of:

* RESTful Backend API (Express.js + TypeScript)
* Modern Web Frontend (Next.js + React)
